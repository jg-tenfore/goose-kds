# 🪿 Goose KDS — Design System

A **React + MUI (Material UI) + Emotion** component library for the TenFore Kitchen
Display System, showcased and customized here in Storybook.

## Structure

The sidebar mirrors the **`eventpipe-prototype-ds`** organization, populated with MUI
components (default-styled for now — styling comes later):

- **Foundations** — Colors, Typography, Spacing, Radius, Border, Effect Styles, Icons, Logos
- **Components** — grouped by purpose: **Actions**, **Navigation**, **Forms**,
  **Feedback & Status**, **Layout & Structure**, **Media & Visuals**, and
  **Typography & Content**
- **App Chrome** — application shell pieces such as Header Navigations and Slideout Menu
- **Account** — Log in, Sign up, Forgot password, Verification screens

*(KDS application screens — organized by the six feature projects — are added under a
**KDS Screens** section as they're built.)*

## Customizing

Everything renders inside `<ThemeProvider theme={theme}>`, so all customization lives in
one place: **`src/theme/theme.ts`**.

- **Global tokens:** `palette`, `typography`, `shape`.
- **Per-component:** `components.MuiButton.styleOverrides` / `defaultProps`, etc.
- **Custom KDS tokens:** `palette.status.*` (normal / warning / late / recalled / priority) —
  see **Foundations → Colors**.

> Uses **core Material UI** plus **MUI X** — Data Grid (Components → Layout & Structure →
> Data Grid) and Date Pickers (Components → Forms → Date Picker). `LocalizationProvider`
> (dayjs) is wired globally.
